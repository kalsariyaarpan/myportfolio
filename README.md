# Arpan's Portfolio

A highly advanced, cinematic, futuristic personal portfolio website built with React.js, featuring cyberpunk aesthetics, smooth animations, and immersive storytelling.

## Features

- **Dark Theme with Neon Accents**: Cyberpunk-inspired design with blue/purple/cyan glow effects
- **Smooth Animations**: Powered by Framer Motion for premium feel
- **3D Elements**: Three.js integration for floating objects and depth
- **Glassmorphism**: Subtle gradients and modern UI elements
- **Responsive Design**: Optimized for mobile and desktop
- **Modular Architecture**: Easy to customize and extend

## Tech Stack

- React.js (TypeScript)
- Tailwind CSS
- Framer Motion
- Three.js
- Vite

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Project Structure

```
src/
├── components/     # Reusable UI components
├── data/          # Static data for skills, projects, etc.
├── App.tsx        # Main app component
└── main.tsx       # Entry point
```

## Customization

- **Skills**: Edit `src/data/skills.ts` to add/remove technologies
- **Projects**: Update `src/data/projects.ts` with your projects
- **Styling**: Modify Tailwind classes in components
- **Content**: Replace placeholder text and images

## Build for Production

```bash
npm run build
```
